Explorando algoritmos de optimización y evolución

¿Qué es Hill Climbing?

Hill Climbing es un algoritmo de búsqueda local utilizado para resolver problemas de optimización.

Parte de una solución inicial y, en cada paso, explora soluciones vecinas buscando una mejora.
Si encuentra una solución mejor, se mueve hacia ella y repite el proceso.
El algoritmo termina cuando no se pueden encontrar soluciones vecinas que mejoren la actual, alcanzando así un óptimo local.

Es simple, eficiente y ampliamente utilizado en problemas donde la búsqueda exhaustiva no es viable.

Enunciado 2 (Optimización del Horario de Estudio)

Un estudiante tiene disponibles 10 franjas horarias libres durante la semana, cada una con una duración y un nivel estimado de productividad (del 1 al 10). El objetivo es seleccionar las mejores franjas para maximizar su productividad semanal total, pero con una restricción: no debe estudiar más de 15 horas en total.

Implementa un algoritmo de hill climbing que, partiendo de una selección aleatoria de franjas horarias, busque mejorar la productividad total sin superar el límite de horas.
Usa Pandas para representar las franjas y NumPy para mutar la solución.

Solución en Python 
import pandas as pd
import numpy as np

# Paso 1: Generar franjas horarias aleatorias
np.random.seed(42)
data = {
    'Duracion': np.random.randint(1, 4, size=10),        # 1 a 3 horas
    'Productividad': np.random.randint(1, 11, size=10)   # 1 a 10
}
franjas = pd.DataFrame(data)
max_horas = 15

# Paso 2: Evaluar una solución
def evaluar(sol, franjas):
    horas = np.sum(franjas['Duracion'] * sol)
    if horas > max_horas:
        return -1
    return np.sum(franjas['Productividad'] * sol)

# Paso 3: Inicializar solución válida
def inicializar_solucion(n):
    while True:
        sol = np.random.randint(0, 2, size=n)
        if np.sum(franjas['Duracion'] * sol) <= max_horas:
            return sol

# Paso 4: Mutar solución
def mutar(sol):
    nueva = sol.copy()
    idx = np.random.randint(len(sol))
    nueva[idx] = 1 - nueva[idx]
    return nueva

# Paso 5: Hill Climbing
def hill_climbing(franjas, max_iter=1000):
    sol = inicializar_solucion(len(franjas))
    prod = evaluar(sol, franjas)
    for _ in range(max_iter):
        nueva = mutar(sol)
        prod_nueva = evaluar(nueva, franjas)
        if prod_nueva > prod:
            sol, prod = nueva, prod_nueva
    return sol, prod

# Ejecutar
sol_opt, prod_opt = hill_climbing(franjas)
print("Franjas horarias:\n", franjas)
print("Solución óptima:", sol_opt)
print("Productividad total:", prod_opt)
print("Total horas:", np.sum(franjas['Duracion'] * sol_opt))

Explicación del Proceso
  • Generación de franjas: Se crean 10 franjas horarias con duración y productividad aleatoria.
  • Solución inicial: Se selecciona aleatoriamente un subconjunto de franjas que no supera las 15 horas.
  • Mutación: Se prueba cambiar una franja (de seleccionada a no seleccionada o viceversa) para buscar una mejor productividad.

  • Resultado: Se obtiene la combinación de franjas que maximiza la productividad sin exceder el límite de horas.
Enunciado 3 (Selección óptima de platos en el menú universitario):

En la cafetería universitaria hay 8 platos diferentes. Cada plato tiene un precio, una cantidad de calorías y una puntuación de satisfacción (según encuestas de estudiantes). Un estudiante solo tiene S/20 para gastar por día y quiere obtener el máximo nivel de satisfacción sin pasarse del presupuesto ni consumir más de 1000 calorías.
Utiliza Pandas para representar el menú y NumPy para generar combinaciones de platos. Aplica hill climbing para encontrar la mejor combinación de 3 platos que maximicen la satisfacción bajo las restricciones indicadas.

Solución en Python 
import pandas as pd
import numpy as np

# Menú de platos: nombre, precio, calorías, satisfacción
menu = pd.DataFrame({
    'Plato': ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H'],
    'Precio':      [5, 7, 4, 6, 3, 8, 2, 5],
    'Calorias': [300, 450, 200, 350, 150, 500, 100, 250],
    'Satisfaccion': [7, 9, 5, 8, 4, 10, 3, 6]
})

presupuesto = 20
max_calorias = 1000
num_platos = 3

# Función de evaluación
def evaluar(sol, menu):
    if np.sum(sol) != num_platos:
        return -1
    total_precio = np.sum(menu['Precio'] * sol)
    total_calorias = np.sum(menu['Calorias'] * sol)
    if total_precio > presupuesto or total_calorias > max_calorias:
        return -1
    return np.sum(menu['Satisfaccion'] * sol)

# Inicializar solución válida
def inicializar_solucion(n):
    while True:
        sol = np.zeros(n, dtype=int)
        idx = np.random.choice(n, num_platos, replace=False)
        sol[idx] = 1
        if (np.sum(menu['Precio'] * sol) <= presupuesto and
            np.sum(menu['Calorias'] * sol) <= max_calorias):
            return sol

# Mutar solución (intercambia un plato seleccionado por uno no seleccionado)
def mutar(sol):
    seleccionados = np.where(sol == 1)[0]
    no_seleccionados = np.where(sol == 0)[0]
    if len(no_seleccionados) == 0:
        return sol.copy()
    nueva = sol.copy()
    quitar = np.random.choice(seleccionados)
    poner = np.random.choice(no_seleccionados)
    nueva[quitar] = 0
    nueva[poner] = 1
    return nueva

# Hill Climbing
def hill_climbing(menu, max_iter=1000):
    sol = inicializar_solucion(len(menu))
    satis = evaluar(sol, menu)
    for _ in range(max_iter):
        nueva = mutar(sol)
        satis_nueva = evaluar(nueva, menu)
        if satis_nueva > satis:
            sol, satis = nueva, satis_nueva
    return sol, satis

# Ejecutar
sol_opt, satis_opt = hill_climbing(menu)
print("Menú:\n", menu)
print("Selección óptima:", sol_opt)
print("Platos elegidos:", menu['Plato'][sol_opt == 1].values)
print("Satisfacción total:", satis_opt)
print("Total gastado:", np.sum(menu['Precio'] * sol_opt))
print("Total calorías:", np.sum(menu['Calorias'] * sol_opt))

Explicación del Proceso
  • Definición del menú: Se crea un menú con 8 platos, cada uno con su precio, calorías y nivel de satisfacción.
  • Solución inicial: Se seleccionan aleatoriamente 3 platos que no superen el presupuesto ni las calorías máximas.

  • Resultado: Se obtiene la combinación de 3 platos que maximiza la satisfacción sin exceder el presupuesto ni las calorías.
Enunciado 4 (Elección de cursos electivos según afinidad y tiempo)

Un estudiante debe elegir 3 de entre 10 cursos electivos posibles. Cada curso tiene una carga horaria semanal, un nivel de interés personal (del 1 al 10), y una calificación histórica promedio (como estimador de facilidad).

El objetivo es encontrar una combinación de cursos que maximice el interés total del estudiante, siempre que la carga horaria combinada no exceda 12 horas semanales.
Usa Pandas para estructurar los datos de los cursos y NumPy para buscar combinaciones óptimas usando un enfoque de hill climbing.

Solución en Python 
import pandas as pd
import numpy as np

# Datos de los cursos: nombre, horas, interés, calificación histórica
cursos = pd.DataFrame({
    'Curso': [f'Curso {i+1}' for i in range(10)],
    'Horas': np.random.randint(2, 6, size=10),  # 2 a 5 horas
    'Interes': np.random.randint(5, 11, size=10),  # 5 a 10
    'Calificacion': np.round(np.random.uniform(12, 20, size=10), 1)  # 12 a 20
})

max_horas = 12
num_elegir = 3

def evaluar(sol, cursos):
    if np.sum(sol) != num_elegir:
        return -1
    total_horas = np.sum(cursos['Horas'] * sol)
    if total_horas > max_horas:
        return -1
    return np.sum(cursos['Interes'] * sol)

def inicializar_solucion(n):
    while True:
        sol = np.zeros(n, dtype=int)
        idx = np.random.choice(n, num_elegir, replace=False)
        sol[idx] = 1
        if np.sum(cursos['Horas'] * sol) <= max_horas:
            return sol

def mutar(sol):
    seleccionados = np.where(sol == 1)[0]
    no_seleccionados = np.where(sol == 0)[0]
    if len(no_seleccionados) == 0:
        return sol.copy()
    nueva = sol.copy()
    quitar = np.random.choice(seleccionados)
    poner = np.random.choice(no_seleccionados)
    nueva[quitar] = 0
    nueva[poner] = 1
    return nueva

def hill_climbing(cursos, max_iter=1000):
    sol = inicializar_solucion(len(cursos))
    interes = evaluar(sol, cursos)
    for _ in range(max_iter):
        nueva = mutar(sol)
        interes_nueva = evaluar(nueva, cursos)
        if interes_nueva > interes:
            sol, interes = nueva, interes_nueva
    return sol, interes

# Ejecutar
sol_opt, interes_opt = hill_climbing(cursos)
print("Cursos:\n", cursos)
print("Selección óptima:", sol_opt)
print("Cursos elegidos:", cursos['Curso'][sol_opt == 1].values)
print("Interés total:", interes_opt)
print("Total horas:", np.sum(cursos['Horas'] * sol_opt))
print("Calificaciones:", cursos['Calificacion'][sol_opt == 1].values)
